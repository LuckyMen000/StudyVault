Часть для фронтенд-разработчика:

Frontend (React):
Разработка пользовательского интерфейса на React, включая компоненты для отображения домашних заданий, формы для добавления и редактирования заданий, и профиль студента.

Взаимодействие с API:
Реализация запросов к API для получения данных о заданиях и их обновления.
Обработка ошибок и отображение сообщений об успешных операциях.

Загрузка и отображение документов:
Разработка компонентов для загрузки и отображения документов, связанных с заданиями.

Маршрутизация:
Настройка маршрутизации для обеспечения навигации между различными страницами сайта.
